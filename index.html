    1、什么是cmd？

    就是在 Windows 操作系统中，利用命令行的方式去操作计算机。

    我们可以利用 cmd 命令去操作计算机，比如：打开文件，打开文件夹，创建文件夹等。

    2、如何打开CMD窗口

    1. 按下快捷键：win + R。

       此时会出现运行窗口。

    2. 在运行窗口中输入 cmd

    3. 回车

    cmd 默认操作 C 盘下的 Users 文件夹下的 XXX 文件夹。（XXX 就是你的用户名）
